# pdfcolorizer
Converts a non colourblind-friendly pdf into a colourblind-friendly pdf

To try out our project: 

In one terminal

```
cd pdfcolorizer
npm install
npm start
```

In another terminal

```
cd server
pip install -r requirement.txt
python3 server.py
```

How to use it?
Upload a file, click 'Process', and enjoy a colourblind-friendly file!
